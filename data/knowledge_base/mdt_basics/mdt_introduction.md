# MDT多学科诊疗基础知识

## 什么是MDT？

MDT（Multi-Disciplinary Team）即多学科诊疗，是指由两个以上相关学科的专家组成工作组，针对某一疾病，通过定期会议形式，提出适合患者的最佳治疗方案，继而由相关学科单独或多学科联合执行该治疗方案。

## MDT的核心特点

1. **多学科协作**：涉及多个医学专业领域的专家
2. **定期会议**：固定时间进行病例讨论
3. **标准化流程**：遵循统一的诊疗流程
4. **患者为中心**：以患者最佳利益为出发点
5. **循证医学**：基于最新医学证据制定方案

## MDT的组成成员

- 肿瘤内科医生
- 肿瘤外科医生
- 放疗科医生
- 病理科医生
- 影像科医生
- 护士
- 药师
- 营养师
- 心理医生

## MDT的诊疗流程

1. 病例筛选与准备
2. 多学科讨论
3. 制定个体化治疗方案
4. 方案执行与跟踪
5. 效果评估与调整

## MDT的优势

- 提高诊断准确性
- 优化治疗方案
- 改善患者预后
- 促进学科交流
- 提升医疗质量
