# MDT医学知识库

这是一个关于MDT（多学科诊疗）的医学垂直知识库，包含MDT的基础知识、诊疗流程、常见肿瘤MDT诊疗和质量评估等内容。

## 知识库结构

### 1. MDT基础知识
- **mdt_introduction.md**: MDT基本概念、核心特点、组成成员、诊疗流程和优势
- **mdt_workflow.md**: MDT诊疗流程与规范，包括病例准备、会议流程和随访管理

### 2. 常见肿瘤MDT诊疗
- **common_cancers_mdt.md**: 肺癌、乳腺癌、结直肠癌、肝癌等常见肿瘤的MDT诊疗要点

### 3. 质量评估与改进
- **mdt_quality_assessment.md**: MDT质量评估指标、持续改进措施和信息化建设

## 使用说明

这个知识库可以作为MDT诊疗的参考资料，帮助医疗工作者：
- 了解MDT的基本概念和流程
- 学习不同肿瘤的MDT诊疗要点
- 掌握MDT质量评估方法
- 提升多学科协作能力

## 文件格式

所有文件均采用Markdown格式，便于阅读和处理。内容结构清晰，包含丰富的医学专业知识。

## 更新维护

建议定期更新知识库内容，根据最新的医学指南和临床实践进行修订。
