# MDT诊疗流程与规范

## 病例准备阶段

### 病例筛选标准
- 复杂疑难病例
- 需要多学科协作的病例
- 新诊断的恶性肿瘤
- 治疗过程中出现新问题的病例

### 资料准备
- 完整的病史资料
- 影像学检查结果
- 病理学报告
- 实验室检查结果
- 既往治疗情况

## MDT会议流程

### 会议前准备
1. 确定会议时间和地点
2. 通知相关科室专家
3. 准备病例资料
4. 确定会议议程

### 会议进行
1. 病例汇报（主管医生）
2. 影像学解读（影像科）
3. 病理学解读（病理科）
4. 各学科意见发表
5. 治疗方案讨论
6. 达成共识

### 会议后工作
1. 记录MDT讨论结果
2. 制定详细治疗方案
3. 与患者及家属沟通
4. 执行治疗方案

## MDT记录与随访

### 记录要求
- 完整的病例讨论记录
- 各学科专家意见
- 最终治疗方案
- 执行时间表

### 随访管理
- 定期评估治疗效果
- 及时调整治疗方案
- 记录不良反应
- 长期生存随访
